# Чек-лист тестирования интернет-магазина Мир Света (UI + функционал)

## Главная страница
- [ ] Отображается логотип
- [ ] Поиск доступен и кликабелен
- [ ] Корректно отображается каталог при наведении
- [ ] Кнопка «Наверх» кликабельна и переносит в начало страницы
- [ ] Кнопка «Личный кабинет» вызывает всплывающее окно для входа

## Регистрация
- [ ] Возможность зарегистрироваться с валидными данными
- [ ] Проверка обязательных полей (ФИО, пароль) 
- [ ] Ошибка при вводе короткого пароля (< 6 символов)

## Вход
- [ ] Возможность войти с существующими данными
- [ ] Ошибка при вводе неверного логина/пароля

## Каталог товаров
- [ ] Категории отображаются правильно, название соответствует фото
- [ ] Возможность открыть страницу товара
- [ ] Фильтр по цене работает корректно
- [ ] Фильтр по цоколю работает корректно
- [ ] Фильтр по бренду работает корректно
- [ ] Сортировка по цене/скидке работает
- [ ] Наличие и кликабельность кнопки «Показать еще»

## Карточка товара
- [ ] Отображается название, артикул, код, цена, фото
- [ ] Кнопка «В корзину» работает
- [ ] Возможность выбрать количество товара
- [ ] В характеристиках есть размеры, вес, технические параметры

## Корзина
- [ ] Товар добавляется в корзину
- [ ] При добавлении товара в корзину, увеличивается счетчик на значке «Корзина»
- [ ] Возможность изменить количество товара/удалить товар
- [ ] При изменении, происходит пересчет итоговой суммы
- [ ] Кнопка «Перейти к оформлению» доступна

- [ ] Возможность выбора способа оплаты

- [ ] Возможность завершить оформление
